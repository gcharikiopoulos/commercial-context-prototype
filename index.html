Customer Data Alignment

A conceptual model showing how aligned customer data enables deterministic billing and invoicing.

Lifecycle

Opportunity (Quote)

Upstream context

Opportunity represents upstream commercial agreement context for the Customer Billing Card (CBC).

Customer Billing Card

Frozen, authoritative commercial snapshot derived from Opportunities.

Project Creation

incl. Linguist Payout Offer

Project creation references an active Customer Billing Card (CBC) and initializes execution.

Usage Capture

Usage Events are recorded against the Project as immutable execution facts.

Billing & Invoicing

Billing derives Charges from Usage Events and the active Customer Billing Card (CBC); Invoicing assembles invoices from Charges.

Linguist Payout Offer

The payout offer is calculated automatically during project setup using expected project revenue from the Customer Billing Card (CBC) and a 75% target contribution margin.
The resulting amount is offered to a linguist.
